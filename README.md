# OvertradeScore

**Am I Overtrading?** — Free self-assessment quiz for traders.

A 2-minute quiz that helps scalpers, intraday traders and swing traders find out if they are overtrading — and what to do about it.

## Live Site
[overtradescore.com](https://overtradescore.com)

## About
OvertradeScore is a free trading psychology tool. Answer 10 questions tailored to your trading style and receive a personalised score and action plan.

Built with HTML, CSS and JavaScript. No frameworks, no dependencies, no tracking.
